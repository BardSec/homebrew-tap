# BardSec Homebrew Tap

Homebrew formulae for BardSec security tools.

## Usage

```bash
brew tap bardsec/tap
```

### PCAP Detective

Desktop threat hunter — analyze packet captures with visual dashboards.

```bash
brew install --cask pcap-detective
```

### Firewall Analyzer

Offline firewall config analyzer — visualize rule conflicts, shadowing, and overly permissive rules.

```bash
brew install --cask firewall-analyzer
```
